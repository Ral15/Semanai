¿Qué hicimos?

Creamos una página web en la que mostramos los estados más felices de México basandonos en el GNH (Gross National Happiness) index del 4to Rey de BhutanJigme Singye Wangchuck.

¿Cómo lo hicimos?

Utilizamos las bases de datos del gobierno, INEGI (Instituto Nacional de Estadística y Geografía), IMSS (Instituto Méxicano del Seguro Social) y la plataforma cívica de datos abiertos datamx.io para sacar información sobre educación, salud, gobierno, seguridad, economía y uso del tiempo de cada estado de México.

El calculo del ranking se realizó utilizando los máximos y mínimos de cada variable propuesta, se definió un rango de separación equitativo entre categorías, desarrollando de esta manera un rango de puntaje para cada una de las áreas. De esta manera los estados acumulan puntos y es posible clasificarlos.
